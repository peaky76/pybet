pybet
=====

pybet is a python-based library of betting utilities. It exists to assist with calculation of bets, stakes and markets

Installation
------------

pybet is available through pypi. Install using your package manager of choice

.. code-block:: python

   pip install pybet

or 

.. code-block:: python

   poetry add pybet
